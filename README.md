# Helicopter

Helicopter game migrated to Unity `6000.5.1f1` and Universal Render Pipeline
`17.5.0`.

Open this repository from Unity Hub with Unity `6000.5.1f1`. The project opens
`Assets/Resources/Scenes/Main.unity` automatically; enter Play mode to run it.

Use WASD or the arrow keys to move. After a collision, press Space to restart.
